# Portfolio V1

An interactive macOS-inspired developer portfolio built using React, Vite, Tailwind CSS, and Framer Motion.

## Live Website

🔗 https://portfolio-v1-mu-sable.vercel.app/

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- JavaScript (ES6+)
- Figma
- Git & GitHub
- Vercel

## Features

- Interactive Spotlight intro animation
- Desktop-inspired portfolio experience
- Folder stack navigation
- Animated project cards with flip interaction
- Scrapbook-style About page
- Embedded resume viewer
- Skills & Certifications page
- Responsive UI
- GitHub and external links
- Smooth page transitions

## Current Projects

### 1. DNA Campus Thrift
A full-stack marketplace for students to buy and sell pre-loved clothing within their campus community.

### 2. Portfolio V1
