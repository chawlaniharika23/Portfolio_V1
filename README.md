# Portfolio V1

An interactive macOS-inspired personal portfolio built using React.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Features

- Animated Spotlight intro
- Interactive folder stack
- macOS-inspired dock and menu bar
- Scrapbook-style About page
- Embedded resume viewer
- Interactive Skills & Certifications page
- Certificate preview modal
- Animated project showcase

## Project Status

Currently in development.
